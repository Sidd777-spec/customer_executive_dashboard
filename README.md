# Executive Dashboard – Retail Customer Behavior Analysis

## Overview
The Executive Dashboard project is a Business Intelligence solution developed to analyze customer shopping behavior in the retail sector. It transforms raw retail transaction data into interactive insights using Python, SQL Server, and Power BI.

The dashboard helps management understand revenue trends, customer demographics, product category performance, seasonal demand, payment preferences, and purchasing patterns.

---

## Objectives
- Analyze customer shopping behavior from historical retail data  
- Identify high-performing product categories  
- Monitor revenue and customer metrics  
- Study customer demographics and purchase trends  
- Support data-driven business decisions through dashboards  

---

## Tools & Technologies Used
- **Python** – Data preprocessing and transformation  
- **Pandas** – Data cleaning and feature engineering  
- **Jupyter Notebook (VS Code)** – Analysis environment  
- **SQL Server (SSMS)** – Database storage and querying  
- **Power BI** – Dashboard development and visualization  
- **GitHub** – Version control and project repository  

---

## Dataset Source
- Retail customer shopping dataset from **Kaggle**

---

## Data Preprocessing
Performed using Python and Pandas:

- Removed duplicate records  
- Checked missing values  
- Standardized column names  
- Converted data formats  
- Created new columns:
  - `age_group`
  - `purchase_frequency_days`
